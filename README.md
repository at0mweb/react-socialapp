
# Social Media Application for Beginners

Made with React, Tailwind and Firebase


## Features

- Login with Google Auth
- Creating posts
- Like posts
- Unlike posts
- More options will be added

## Installation

Git clone my project

```bash
  git clone https://github.com/at0mweb/react-socialapp.git
  cd react-socialapp
  npm install
  npm start
```
